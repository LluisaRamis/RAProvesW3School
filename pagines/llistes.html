ETIQUETES INLINE I ELEMENTS DE BLOC

  1. Etiquetes inline:

    <a> (Enllaç): Utilitzada per crear hiperenllaços a altres pàgines web o a punts dins de la mateixa pàgina.
    <span> (Interval): S'utilitza per aplicar estils a una part específica del text sense interrompre el flux del document.
    <em> (Èmfasi): Emfatitza el text, generalment mostrant-lo en cursiva, per indicar una major importància.
    <strong> (Rellevància): Indica que el text és important, normalment mostrant-lo en negreta.
    <img> (Imatge): Insereix una imatge dins del document HTML.
    <code> (Codi): Destaca un fragment de codi dins del text, generalment mostrant-lo en una font monospace.
    <kbd> (Tecla): Representa una entrada de l'usuari via teclat, mostrant les tecles que s'han de prémer.
    <mark> (Marca): Ressalta una part del text, semblant a l'efecte de subratllar amb un marcador de color.
    <sup> (Superíndex): Mostra text en superíndex, elevant-lo per sobre de la línia base del text normal.
    <sub> (Subíndex): Mostra text en subíndex, baixant-lo per sota de la línia base del text normal.

  2. Els elements de Bloc:

    <div> (Divisió): S'utilitza per contenir altres elements HTML i aplicar estils o scripts a un conjunt de contingut comú.
    <p> (Pàragraf): Representa un paràgraf de text.
    <h1> a <h6> (Encapçalament): Utilitzats per definir encapçalaments, amb <h1> sent el més important i <h6> el menys important.
    <ul> (Llista desordenada): Crea una llista d'elements on l'ordre no és important.
    <ol> (Llista ordenada): Crea una llista d'elements on l'ordre sí és important.
    <li> (Element de llista): Defineix un element dins d'una llista, ja sigui ordenada o desordenada.
    <blockquote> (Cita bloc): Utilitzat per representar una secció citada d'una altra font.
    <pre> (Text preformatat): Mostra text amb espais en blanc, salts de línia i altres elements tal com es defineixen en el codi HTML.
    <table> (Taula): Crea una taula per organitzar dades en files i columnes.
    <section> (Secció): Defineix una secció de contingut dins d'un document, com un capítol, capítol d'una pàgina web o qualsevol altre segment de contingut.
