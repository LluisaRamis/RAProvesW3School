1. Etiquetes inline:

  2. <a> (Enllaç): Utilitzada per crear hiperenllaços a altres pàgines web o a punts dins de la mateixa pàgina.
  3. <span> (Interval): S'utilitza per aplicar estils a una part específica del text sense interrompre el flux del document.
  4. <em> (Èmfasi): Emfatitza el text, generalment mostrant-lo en cursiva, per indicar una major importància.
  5. <strong> (Rellevància): Indica que el text és important, normalment mostrant-lo en negreta.
  6. <img> (Imatge): Insereix una imatge dins del document HTML.
  7. <code> (Codi): Destaca un fragment de codi dins del text, generalment mostrant-lo en una font monospace.
  8. <kbd> (Tecla): Representa una entrada de l'usuari via teclat, mostrant les tecles que s'han de prémer.
  9. <mark> (Marca): Ressalta una part del text, semblant a l'efecte de subratllar amb un marcador de color.
  10. <sup> (Superíndex): Mostra text en superíndex, elevant-lo per sobre de la línia base del text normal.
  11. <sub> (Subíndex): Mostra text en subíndex, baixant-lo per sota de la línia base del text normal.

    12. Els elements de Bloc:

  12. <div> (Divisió): S'utilitza per contenir altres elements HTML i aplicar estils o scripts a un conjunt de contingut comú.
  13. <p> (Pàragraf): Representa un paràgraf de text.
  14. <h1> a <h6> (Encapçalament): Utilitzats per definir encapçalaments, amb <h1> sent el més important i <h6> el menys important.
  15. <ul> (Llista desordenada): Crea una llista d'elements on l'ordre no és important.
  16. <ol> (Llista ordenada): Crea una llista d'elements on l'ordre sí és important.
  17. <li> (Element de llista): Defineix un element dins d'una llista, ja sigui ordenada o desordenada.
  18. <blockquote> (Cita bloc): Utilitzat per representar una secció citada d'una altra font.
  19. <pre> (Text preformatat): Mostra text amb espais en blanc, salts de línia i altres elements tal com es defineixen en el codi HTML.
  20. <table> (Taula): Crea una taula per organitzar dades en files i columnes.
  21. <section> (Secció): Defineix una secció de contingut dins d'un document, com un capítol, capítol d'una pàgina web o qualsevol altre segment de contingut.
